<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| CLI → filesystem (config dir) | pane.yaml + identity.age hold secrets; perms must be 0600 |
| CLI → git remote | backup pushes encrypted blob to a user-owned private repo |
| pane.yaml.age → public/shared surface | encrypted blob is the only thing that may travel |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-PERM | Information Disclosure | pane.yaml / identity.age | mitigate | os.OpenFile O_CREATE\|O_WRONLY\|O_TRUNC, 0600; tests assert Perm()==0600 (Pitfall 4) |
| T-03-CRYPTO | Information Disclosure | backup encryption | mitigate | filippo.io/age X25519 only; no hand-rolled crypto (V6); cross-identity decrypt-fails test |
| T-03-PLAINTEXT | Information Disclosure | git push payload | mitigate | Only pane.yaml.age (ciphertext) is added/committed/pushed; plaintext pane.yaml never staged |
| T-03-GITARG | Tampering | git subprocess | mitigate | Runner.Run("git", args...) variadic, no `sh -c`; user remote/path passed as discrete args |
| T-03-ESCROW | Repudiation | key recovery | accept | No escrow / no central service by design (invariants 4/5/7); recovery limitation documented |
</threat_model>
