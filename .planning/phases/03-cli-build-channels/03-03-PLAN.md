<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| CLI → translate/docker subprocess | argv + env (incl. epoch) cross to external binaries |
| private pane → injection artifact | secrets leave HOME only as a LOCAL tar, never inside the shared ISO |
| private-pane dst paths → target filesystem | attacker-influenced paths could escape extraction root |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-DKARG | Tampering | docker/translate invocation | mitigate | Runner.Run(name, args...) variadic; no `sh -c`; image ref + mounts + env as discrete args |
| T-03-LEAK | Information Disclosure | private-injection.tar placement | mitigate | tar written to --out (next to ISO); test asserts no secrets under arch-profile tree (Pitfall 3) |
| T-03-TRAV | Tampering | injection dst paths | mitigate | _sanitize_dst-equivalent rejects absolute + `..` (T-02-08); negative tests |
| T-03-EPOCH | Tampering | determinism keying | mitigate | single derivation from resolved.json sha256; consistency assert vs manifest.py value |
</threat_model>
