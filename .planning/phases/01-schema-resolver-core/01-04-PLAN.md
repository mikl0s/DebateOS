| sig_level=Never repo declarations | Opinion-declared unsigned repos (data); must be surfaced, never silently applied |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-01-11 | Repudiation | a resolution decision with no explanation | mitigate | every Applied/Skipped/Dropped/conflict decision carries a structured Explanation (Rule field); test coverage enforces |
| T-01-12 | Tampering | non-deterministic canonical JSON leaking map order | mitigate | struct-based output, sorted intermediate map keys, no float64; determinism test across -count=5 |
| T-01-13 | DoS | resolve loop on pathological conflict graph | accept | bounded by opinion count (speeches small); toposort is O(E log V); no untrusted unbounded recursion |
