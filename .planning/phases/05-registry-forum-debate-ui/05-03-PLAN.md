<objective>
Build the Forum's storage + read API core (FORM-01/02/03): a `Store` interface backed by sqlc-generated queries over modernc.org/sqlite (PINNED v1.46.1, from 05-01's go.mod) with FTS5 search, subscription edges, and OAuth-identity-gated ratings — all tested against in-memory SQLite. No native accounts — ratings/subscriptions key on a GitHub user ID string supplied by the (future) OAuth layer; this plan accepts the userID as a parameter and a fake identity in tests.
